Olam Live Prices in Excel

Stream real-time commodity data directly into your cells. No clicking refresh.

Part 1: One-Time Setup

1

Get the Configuration File

Download the add_in.xml configuration file to your computer below:

Download add_in.xml

⛔ DO NOT OPEN THIS FILE directly in Excel. Save it to your computer first.

2

Upload to Excel

Open Excel Online. Go to the Insert tab on the ribbon > Click Add-ins (or My Add-ins) > Upload My Add-in > Select the add_in.xml file you just downloaded.

Part 2: The Magic Formula

The Add-in works entirely in the background. Just type this formula into any blank cell:

=OLAM.LIVEPRICE("ticker", "field")

1. Valid Tickers

Use exact Bloomberg IDs or internal API keys (Case-insensitive):

USDJPY CCH6 cc1 usd-jpy

2. Valid Fields

The data points you can pull:

MID BID ASK LAST_PRICE VOLUME OPEN HIGH LOW

Real Examples in Action

=OLAM.LIVEPRICE("cc1", "MID")

=OLAM.LIVEPRICE("USDJPY", "ASK")

Note: The cell might temporarily say #BUSY! or Waiting... while it connects. It will then display the live streaming price!
